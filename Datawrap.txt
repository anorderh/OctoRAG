Datawrap

Goals
- Allow users to search datasets and bring them into client
- Allow users to create users & manage credentials
- Allow users to save workspaces
- Allow users to render 2D graphs based on datasets
- Allow users to render 3D graphs (minimal) based on datasets
- Allow users to query datasets using SQL
- Allow users to efficiently manage datasets & imported data

Stack
- React (Typescript, RxJS)
- NodeJS
- MongoDB
- Azure

Libraries
- kaggle-node
- React, Typescript, RxJS
- NodeJS, axios, Azure SDK
- MUI, MUI Library
- ThreeJS
